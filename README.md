# Blazor WebAssembly Layout Template

A reusable **Blazor WebAssembly project template** designed to give you a clean, structured, and ready-to-use project layout.  
After installing the template, you can create new Blazor projects directly from **Visual Studio** or using the **.NET CLI**.
